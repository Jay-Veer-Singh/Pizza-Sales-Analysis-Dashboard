# Pizza-Sales-Analysis-Dashboard

### Analyzed pizza sales data to evaluate business performance using SQL and Power BI. The project focused on calculating key sales KPIs, identifying sales trends, and understanding product performance across categories and sizes.

### Performed data analysis using SQL to compute metrics such as total revenue, average order value, total orders, and pizzas sold. Analyzed daily, hourly and monthly patterns to identify peak business periods and customer ordering behavior.

### Built an interactive Power BI dashboard to visualize KPIs, sales trends, and top and bottom-performing pizzas by revenue, quantity, and order volume. The dashboard enables quick filtering and supports data-driven decision-making for business stakeholders.

### Tools Used
MySQL (Data analysis & querying),
Power BI (Dashboard & visualization)
